## Installation

Install the library via npm:

```bash
npm install my-angular-library
```

## Usage

Import the library in your Angular app:

```typescript
import { MyLibraryModule } from 'my-angular-library';

@NgModule({
  imports: [MyLibraryModule],
  // ...
})
export class AppModule {}
```
